About This Course

Sustainable Logistics Operations - Learning through practical optimization

Course Overview

This course provides hands-on experience with optimization techniques applied to sustainable logistics operations. Through a series of practical exercises, students learn to solve real-world logistics problems using modern optimization tools and algorithms.

Topics Covered

  • Constraint Programming: Using CP-SAT solver for combinatorial optimization
  • Linear Programming: Solving linear optimization problems
  • Assignment Problems: Optimal resource allocation
  • Routing Problems: Traveling Salesman Problem (TSP) and variants
  • Vehicle Routing: VRP, CVRP, and extensions
  • Packing Problems: Knapsack problem, bin packing
  • Facility Location: P-median and set covering problems

Learning Approach

Each exercise is designed to be:

  • Practical: Real code that students can run and modify
  • Accessible: Available through Google Colab without installation
  • Educational: Includes explanations of concepts and approaches
  • Progressive: Builds from simple to complex problems
